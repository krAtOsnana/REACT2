*> hooks introduced in react after version 16.8

* react Hooks: special function that allows functional component
* to use react feacture like state management and lifecycle method
* without writting class component

*>in calss component we have a state management methods and lifecycle methods
but in functional component we cannot use such feature directly, we need to use
hooks for achieving such functionalities
*>hooks introduced in react to avoide writing class component.

# Rules to use Hooks :

1.must import hooks from react ( import React,{useState} from "react"; );
2.can only be calleed/use inside the functional component;
3.can only be called at the top level of the functional component;
4.cannot be used in conditional statement;
